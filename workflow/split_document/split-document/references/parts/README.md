一个源代码文件可能包含以下部分：

- 代码引入
- 宏
- 函数
- enum
- 结构体数据类型或实例

意味着，一个源代码里面，可能存在以上类型的代码片段。

# 一些语言里的这些部分

## Python

代码引入：见 `.\import\import.html` ，其中[import语句](.\import\import-statement.md)非常有可能是非常经典的代码引入。

## C++

- 引入头文件（代码引入）：`#include <头文件名>` 或 `#include "头文件名"` 
- 函数：见 `.\cppreference.cn-C++-functions.md`
